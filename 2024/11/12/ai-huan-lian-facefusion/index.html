ai换脸facefusion

克隆仓库
git clone https://github.com/facefusion/facefusion-docker.git
进入目录facefusion
sudo docker compose -f docker-compose.cpu.yml up
完成之后，浏览器打开
http://localhost:7865

经常会出错停止，加上自动重启
sudo docker update --restart=unless-stopped facefusion-docker-facefusion-cpu-01
sudo docker images
