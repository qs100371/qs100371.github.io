安卓手机上的linux模拟器termux

Termux是一个适用于 Android 的终端模拟器，其环境类似于 Linux 环境。 无需Root或设置即可使用。

安装

官网：https://termux.dev/cn/
下载：https://github.com/termux/termux-app/releases

手机上下载对应的apk安装，打开显示：

使用

访问外部存储
termux-setup-storage
安装ssh
pkg install -y openssh
设置密码
passwd
查看用户名
whoami
启动ssh服务端
sshd
现在可以连接ssh了
ssh User@Host -p 8022
设置打开应用时自动启动ssh服务
echo "sshd" >> ~/.bashrc
