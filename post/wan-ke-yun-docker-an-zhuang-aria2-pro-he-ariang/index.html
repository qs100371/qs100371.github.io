玩客云docker安装Aria2 Pro和AriaNg

2024-11-22 1 min read # onecloud # docker

Aria2是一个命令行下载器。

安装Aria2 Pro

docker run -d  \
--name aria2-pro  \
--restart unless-stopped  \
--log-opt max-size=1m  \
-e PUID=$UID  \
-e PGID=$GID  \
-e UMASK_SET=022 \
-e RPC_SECRET=qq1172851433  \
-e RPC_PORT=6800  \
-p 6800:6800 \
-e LISTEN_PORT=6888 \
-p 6888:6888  \
-p 6888:6888/udp \ 
-v /data/aria2:/config \
-v /data/downloads:/downloads \
p3terx/aria2-pro

RPC_SECRET可自定

安装AriaNg

docker run -d \
--name ariang  \
--log-opt max-size=1m \
--restart unless-stopped  \
-p 6880:6880  \
--network host  \
p3terx/ariang

配置Aria2

浏览器中输入192.168.0.2:6880(玩客云armbain地址)，打开AriaNg的管理页。
不用管认证失败的弹窗，选择左侧“系统设置"下的"AriaNg设置"，在右侧选择“RPC()”，填上创建容器时填的端口和密钥。

aria2下载比不上qbittorrent，暂时没有发现有什么更好的用途。
