玩客云Docker搭建私有笔记服务Trilium

2024-11-25 1 min read # onecloud

Trilium，是一个基于 electron、跨平台的分层级笔记应用，主要使用场景是建立个人知识库。

安装

mkdir -p /data/trilium/data
docker run -d \
  --name trilium \
  --restart=always \
  --log-opt max-size=1m \
  -v /data/trilium/data:/root/trilium-data \
  -e TRILIUM_DATA_DIR=/root/trilium-data \
  -p 8181:8080 \
  nriver/trilium-cn:latest
