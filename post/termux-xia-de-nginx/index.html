termux下的nginx

2025-01-10 1 min read # termux # web

手机也可以做为一个小服务器。termux+nginx

安装nginx

pkg install nginx

运行

nginx

echo $PREFIX
/data/data/com.termux/files/usr
cd $PREFIX/share/nginx/html/
.../nginx/html $
把文件放在目录下。
网页从ip:8080访问。
