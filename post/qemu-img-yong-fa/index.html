qemu-img用法

2025-01-08 1 min read # linux

qemu-img可以用来缩减虚拟磁盘映像的大小。
qemu-img convert -O qcow2 -c disk.qcow2 disk-new.qcows

创建磁盘映像

qemu-img create -f qcow2 -o size=10G ubuntu.img
qemu-img create -f vmdk -o size=20G ubuntu.img

增加映像大小

qemu-img resize ubuntu.qcow2 +5GB

获取磁盘映像信息

qemu-img info alpine.img
