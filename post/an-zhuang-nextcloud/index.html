安装nextcloud

2024-11-01 1 min read # web

nextcloud是一个功能强大的网盘系统，snap安装。

安装
sudo apt install snapd
sudo snap install nextcloud
snap changs nextcloud
sudo snap start nextcloud

配置
设置nextcloud账号密码
sudo nextcloud.manual-install 你的用户名 你的密码
设置成功后便可在nextcloud主机上使用浏览器通过http//:127.0.0.1 进行访问
端口修改
sudo snap set nextcloud ports.http=9999
设置nextcloud不限制访问方式
sudo nextcloud.occ config:system:set trusted_domains 1 --value=*.*.*.*
重启nextcloud
sudo snap restart nextcloud
注意：nextcloud.manual-install和nextcloud.occ可能要带路径/snap/bin/
