Ubuntu上修改系统时区的方法

2025-01-24 1 min read # linux

正确的时区设置对于确保准确的时间和日期非常重要，本文介绍两种常用的方法来修改Ubuntu系统的时区设置：使用timedatectl命令和直接编辑/etc/timezone文件。

使用timedatectl命令

sudo timedatectl set-timezone Asia/Shanghai
验证更改：
timedatectl

               Local time: Fri 2025-01-24 11:42:06 CST
           Universal time: Fri 2025-01-24 03:42:06 UTC
                 RTC time: Fri 2025-01-24 03:42:07
                Time zone: Asia/Shanghai (CST, +0800)
System clock synchronized: no
              NTP service: active
          RTC in local TZ: no

修改/etc/timezone文件

在文件中写入“Asia/Shanghai"
更新系统时间：重启或
sudo systemctl restart systemd-timesyncd
