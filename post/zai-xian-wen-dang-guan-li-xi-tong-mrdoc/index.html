在线文档管理系统mrdoc

2025-02-19 1 min read # web

MrDoc 觅思文档，自托管、私有部署的在线文档管理系统和知识库。可以创建各类私有化部署的文档应用，进行知识管理、构建团队文库、制作产品手册以及在线教程等。

项目地址：https://www.mrdoc.pro/
文档：https://mrdoc.fun/doc/18/

部署方式

docker run -d --name mrdoc -p 10086:10086 jonnyan404/mrdoc-alpine

默认用户：admin
默认密码:执行 docker logs mrdoc 2>&1|grep pwd 获取
从IP:10086 访问。
