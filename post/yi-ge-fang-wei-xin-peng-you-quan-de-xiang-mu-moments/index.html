一个仿微信朋友圈的项目moments

2025-01-24 1 min read # web

moments是一个仿微信朋友圈的项目，适合记录生活。本文记录了部署方法。

拉取镜像

docker pull kingwrcy/moments:latest

编辑docker-compose.yml

内容如下

services:
  moments:
    image: kingwrcy/moments:latest
    ports:
      - "80:3000"  # 左边的 8080 可以随意更改成服务器上没有用过的端口
    volumes:
      - /data/moments:/app/data
      - /etc/localtime:/etc/localtime:ro
      - /etc/timezone:/etc/timezone:ro

建立目录
mkdir -p /data/moments

启动容器

docker-compose up -d
从ip:80访问，默认用户名密码：admin/a123456，登录进去后，后台可以自己修改密码。

此项目没有armv7的镜像，不能运行在玩客云上。
