ffmpeg合并多个MP4视频

2024-11-27 1 min read # web

FFmpeg 是一个强大的多媒体框架，可以处理音频和视频文件。

直接将多个MP4直接拼接成一个MP4

  • 将合并的文件夹都放在同一个文件夹下，例如：test.mp4，test2.mp4

  • 新建一个video.txt文件，文件的内容是需要合并的视频名称，内容格式为
    file 'test.mp4'
    file 'test2.mp4'

  • 执行ffmpeg

ffmpeg -f concat -i video.txt -c copy concat.mp4
执行完就可以看到合并起来的视频concat.mp4。

ffmpeg下载：https://ffmpeg.org
