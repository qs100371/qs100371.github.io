在线办公套件onlyoffice

2025-02-27 1 min read # web

ONLYOFFICE Document Server 是一款在线办公套件，由文本、电子表格和演示文稿的查看器和编辑器组成，与 Office Open XML 格式完全兼容：.docx、.xlsx .pptx，并支持实时协作编辑。

官网：https://www.onlyoffice.com/zh/

部署

单独安装 ONLYOFFICE 文档服务器：
docker run -i -t -d -p 80:80 onlyoffice/documentserver
