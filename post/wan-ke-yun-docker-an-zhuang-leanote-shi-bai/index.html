玩客云docker安装leanote失败

2024-11-24 1 min read # onecloud # web

蚂蚁笔记 = 笔记 + 博客 + 协作 + 私有云

安装

mkdir -p /data/leanote/db
mkdir -p /data/leanote/conf
mkdir -p /data/leanote/files
mkdir -p /data/leanote/public/upload

docker run -d --name leanote \
    -v /data/leanote/db:/data/db \
    -v /data/leanote/conf/:/data/leanote/conf \
    -v /data/leanote/files:/data/leanote/files \
    -v /data/leanote/public/upload:/data/leanote/public/upload \
    -p 9000:9000 --restart unless-stopped \
    axboy/leanote

玩客云是armv7，镜像是amd64，不能运行。加上--platform linux/amd64也不行。在debian12虚拟机里可以正常部署。
