docker安装音乐服务navidrome

2024-11-18 1 min read # docker

navidrome是一个本地音乐服务。

Navidrome

docker架设本地服务

docker run -d \
   --name navidrome \
   --restart=unless-stopped \
   --user $(id -u):$(id -g) \
   -v /data/music:/music \
   -v /data/navidrome:/data \
   -p 4533:4533 \
   -e ND_LOGLEVEL=info \
   deluan/navidrome:latest

成功后，访问localhost:4533。

音乐：

歌曲宝 https://www.gequbao.com/
可单曲下载，怕麻烦直接下打包曲库，要转迅雷云下载，每月有5G高速流量，剩下的只能1Mbps慢慢下了。音乐文件的ID3 tag都没有了，可以用Music Tag来更新，同时有歌词和封面信息，win版本下载。
