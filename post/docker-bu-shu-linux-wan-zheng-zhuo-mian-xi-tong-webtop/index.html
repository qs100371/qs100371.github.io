docker部署linux完整桌面系统webtop

2025-02-15 1 min read # linux

WebTop是一个基于 Linux （ Ubuntu 和 Alpine 两种版本）的轻量级容器，具有在浏览器中运行的完整桌面环境，具有基本的窗口管理器、像素完美的渲染分辨率、音频支持、剪贴板支持、屏幕键盘支持，以及用于上传/下载的远程文件管理。

项目地址：linuxserver/docker-webtop

功能特色：

  • 像素渲染的完美分辨率
  • 基本的窗口管理器
  • 音频支持
  • 使用 RDP 协议
  • 剪贴板支持
  • 屏幕键盘支持
  • 远程文件上传/下载管理

cli部署

docker run -d \
  --name=webtop \
  --privileged `#optional` \
  -e PUID=1000 \
  -e PGID=1000 \
  -e TZ=Asia/Shanghai \
  -e SUBFOLDER=/ `#optional` \
  -p 83:3000 \
  -v /data/webtop:/config \
  -v /var/run/docker.sock:/var/run/docker.sock `#optional` \
  --shm-size="2gb" `#optional` \
  --restart unless-stopped \
  linuxserver/webtop

从ip:83访问。


支持x86-64，arm64，不支持armhf（也就是armv7）。
由tag决定linux版本，latest是xfce-alpine。
