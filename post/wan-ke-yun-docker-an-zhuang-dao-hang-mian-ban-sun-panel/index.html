玩客云docker安装导航面板sun-panel

2024-11-26 1 min read # onecloud # web

onecloud上部署的服务多了，有的端口就记不太清了，用Sun-Panel做一个导航面板

安装

mkdir -p /data/sun-panel
docker pull hslr/sun-panel:latest
docker run -d --restart=always -p 8000:3002 \
-v /data/sun-panel/conf:/app/conf \
-v /var/run/docker.sock:/var/run/docker.sock \
--name sun-panel \
hslr/sun-panel:latest

打开192.168.0.2:8000端口访问

默认账号密码
账号：admin@sun.cc
密码：12345678


重置密码

docker exec -it sun-panel bash
./sun-panel -password-reset
输出
The password has been successfully reset. Here is the account information
Username admin@sun.cc
Password 12345678
