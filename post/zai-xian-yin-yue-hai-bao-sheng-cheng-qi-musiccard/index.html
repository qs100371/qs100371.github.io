在线音乐海报生成器musiccard

2025-02-27 1 min read # web

MusicCard 是一个基于 Next.js 开发的在线音乐海报生成工具,让用户能够为自己喜爱的网易云音乐创建精美的可视化海报。

项目地址：aidaox/MusicCard

🎨 支持多种海报主题和布局
🎵 支持网易云音乐链接解析
📱 响应式设计,支持移动端
🌈 自定义海报样式和颜色
📝 支持歌词展示
💾 支持导出高清图片

拉取镜像

docker pull wbsu2003/musiccard:v1

开启容器

docker run -d \
   --restart unless-stopped \
   --name musiccard \
   -p 8940:3000 \
   wbsu2003/musiccard:v1

从ip:8940打开。
