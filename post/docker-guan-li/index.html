Docker管理

2024-11-07 1 min read # docker

安装dock.io
启动服务
systemctl enable docker

镜像操作

命令 作用
docker search 关键字 搜索镜像
docker pull 仓库名称 获取镜像
docker images 仓库名称 查看镜像信息
docker rmi 仓库名称:镜像ID号 删除镜像

容器操作

命令 作用
docker create 镜像 运行的程序 创建容器
docker start 容器的ID/名称 运行容器
docker stop 容器的ID/名称 停止容器
docker rm 容器的ID/名称 删除容器
docker ps 查看容器
docker exec 容器ID/名称 /bin/bash 进入容器

停止openwrt
docker stop openwrt
停用自动重启功能
docker update --restart=no openwrt
在Docker中，容器的自动重启行为是由Docker引擎的重启策略控制的。Docker引擎提供了以下几种重启策略：

  • no：不重启容器，默认策略
  • always：容器退出时总是重启
  • on-failure：容器退出时，只有在退出状态非零时才重启
  • unless-stopped：除非手动停止容器，否则总是重启
