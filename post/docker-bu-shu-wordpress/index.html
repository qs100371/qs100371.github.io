docker部署wordpress

2025-01-12 1 min read # web

WordPress是一个以PHP和MySQL为平台的自由开源的博客软件和内容管理系统。WordPress具有插件架构和模板系统，是目前因特网上最流行的博客系统。本文用docker部署wordpress。

部署wordpress

docker pull wordpress
mkdir /data/wordpress
docker run -it --name wordpress -p 9999:80 \
-v /data/wordpress:/var/www/html -d wordpress

建立数据库

本地已经有了mysql，直接使用：

docker exec -it mysql bash
mysql -u root -p
create database wordpress;
show databases;
exit


网站设置

在浏览器打开IP:9999，进入网站设置，填上数据库信息，完成部署。

有了数据库，再部署这类网站就容易了。主机主要是受限于玩客云是armv7，许多应用不支持。
