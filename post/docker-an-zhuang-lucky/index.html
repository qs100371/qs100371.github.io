docker安装lucky

2024-12-10 1 min read # web

lucky是一个内网工具，功能很多。

安装

host模式, 同时支持IPv4/IPv6, Liunx系统推荐
docker run -d --name lucky --restart=always --net=host gdy666/lucky
挂载主机目录, 删除容器后配置不会丢失。可替换 /data/lucky 为主机目录, 配置文件为lucky.conf
docker run -d --name lucky --restart=always --net=host -v /data/lucky/luckyconf:/goodluck gdy666/lucky

通过lucky可实现内网穿透，超级复杂。lucky设置里可导出配置文件，这是一个zip文件，解压后放到luckyconf文件夹里可恢复设置。
