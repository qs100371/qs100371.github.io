FileCodeBox: 一款轻量级的文件快递柜

2025-02-07 1 min read # web

在日常生活和工作中，经常需要将一些文件或文本分享给他人，或者在不同的设备之间传递信息。然而，许多共享工具都需要下载特定的应用程序或使用复杂的登录过程才能完成这些操作，这会给我们带来很多不必要的麻烦。那么这款工具FileCodeBox，正好可以解决这样的问题。

拉取镜像

docker pull lanol/filecodebox:beta

开启容器

mkdir /data/filecodebox
docker run -d --restart=unless-stopped \
-p 12345:12345 -v  /data/filecodebox/:/app/data \
--name filecodebox lanol/filecodebox:beta

从IP:12345访问。


后台管理

IP:12345/#/admin，密码：FileCodeBox2023，进入后台可以更改密码和上传文件大小限制。

注意这个地址，官方说的/admin不能正常访问。
